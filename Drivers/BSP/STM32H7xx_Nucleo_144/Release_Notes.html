Update History

Main Changes

  • Add support for NUCLEO-H743ZI2 (MB1364 board).
    • Changes to support the MB1364 board are under the compilation define USE_STM32H7XX_NUCLEO_144_MB1364
  • Add support for NUCLEO-H745ZI-Q (MB1363 board).
    • Changes to support the MB1363 board are under the compilation define USE_STM32H7XX_NUCLEO_144_MB1363
  • Fix LCD_IO_WriteMultipleData function to handle odd number of bytes.
  • Update ADC configuration (according to latest HAL ADC updates) to handle the Adafruit shiled joystick.

  • Align Nucleo_144 BSP driver to the latest HAL update.
