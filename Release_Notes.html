Purpose

The STM32H7 Matter Bridge (STM32 Matter protocol software for STM32CubeIDE) - provides a port of the Matter protocol to the supported boards

    • AN4891 - STM32H72x, STM32H73x, and single-core STM32H74x/75x system architecture and performance

    • ES0396 - STM32H750xB and STM32H753xI device limitations

Update History

Main Changes

1st beta release containing a Matter bridge for non-matter devices.

It runs on NUCLEO-H753ZI-Q board Rev.C with Ethernet connectivity.

Known Limitations

  • Matter Test Harness not run on this example.

  • Numerous compilation warnings, mostly in the Matter SDK, in LwIP. Some compilation warnings are in the application but are just unused arguments.

  • The upstream application build is supported on Windows, MacOS and Linux. However the package itself has only been tested on a Windows 10 host.

Development Toolchains and Compilers

  • STM32CubeIDE, version 1.9.0 with gnu 11 and gnu++ 14 toolchain + ST-Link

Supported Devices and Boards
